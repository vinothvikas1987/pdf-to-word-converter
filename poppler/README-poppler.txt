Poppler for Windows
====================
This folder contains Poppler binaries (GPL-licensed) used by pdf2image
to convert PDF pages to images.

Source: https://github.com/oschwartz10612/poppler-windows/releases
License: GNU GPL v2 or later (https://www.gnu.org/licenses/gpl-2.0.html)

These binaries are included as-is for user convenience. You may also
download them separately from the URL above.
